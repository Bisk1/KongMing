1) Mysql
a) odnow baze (skasuj i utworz)
DROP DATABASE IF EXISTS chinese_db;
CREATE DATABASE IF NOT EXISTS chinese_db
CHARACTER SET = utf8
COLLATE = utf8_bin;
GRANT ALL ON chinese_db.* TO 'chinese_user'@'localhost' IDENTIFIED BY 'chinese_pass';

2) Synchronizuj baze
python manage.py syncdb

3) Restore dump - musisz byc w folderze z plikiem dumpa
mysql -u root chinese_db < chinese-dump.sql

4) Create dump (jesli potrzebne)
mysqldump -u root chinese_db > chinese-dump.sql
